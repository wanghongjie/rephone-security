## 📲 安装说明

### Android设备安装
1. **下载APK**: 选择 `RePhone-Security-release.apk`
2. **允许未知来源**: 设置 > 安全 > 允许安装未知来源应用
3. **安装应用**: 点击APK文件进行安装
4. **启动应用**: 安装完成后点击"打开"

### 开发者安装
```bash
# 使用ADB安装
adb install RePhone-Security-release.apk

# 或者使用Flutter命令
flutter install
```
